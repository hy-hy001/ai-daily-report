# 🤖 AI 日报精选 TOP 10

每日自动生成，基于 [AIHOT](https://aihot.virxact.com) 精选评分排行。

## 使用方法

### Obsidian 同步

1. 安装 [Obsidian Git](https://github.com/denolehov/obsidian-git) 插件
2. 将本仓库 clone 到 Obsidian 仓库目录下
3. 插件自动拉取每日更新

### 手动查看

直接在 [GitHub](https://github.com/hy-hy001/ai-daily-report/blob/main/AI%E6%97%A5%E6%8A%A5.md) 上查看。

## 更新频率

每天北京时间 06:03 自动更新。
